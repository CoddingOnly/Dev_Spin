Веб-разработка

(HTML, CSS, JavaScript)

ВВЕДЕНИЕ

Прежде чем начинать изучение HTML, CSS и JavaScript, важно понять общую картину:
• Веб-разработка — это процесс создания веб-сайтов и веб-приложений.
• Веб-разработка состоит из Frontend (клиентская часть) и Backend (серверная часть).
• HTML, CSS и JavaScript — это языки, которые используются для создания интерфейсов и взаимодействия с пользователем.

ОСНОВЫ HTML

HTML — это язык разметки, который нужен для создания структуры веб-страницы.

СТРУКТУРА HTML-ДОКУМЕНТА

!DOCTYPE html — объявление типа документа.

html lang="ru" — корневой элемент, язык страницы.

head — метаинформация о странице (не отображается на странице)..

body — тело страницы (содержит контент).

ОСНОВНЫЕ ЭЛЕМЕНТЫ HTML

ФОРМЫ

Формы — это элементы HTML, которые позволяют пользователю отправлять данные на сервер:
form action="/submit" method="post" label for="username"
Имя label input type="text" id="username" name="username"> input type="submit" value="Отправить" form

ОСНОВЫ CSS

CSS используется для стилизации HTML. Вы можете задавать цвета, шрифты, размеры и другие параметры.

Включение CSS в HTML

• Внутренний стиль: <style>...</style> внутри тега <head>
• Внешний файл: <link rel="stylesheet" href="styles.css">

Селекторы CSS

Селекторы определяют, к каким элементам HTML применяется стиль
• По тегу: p { color: red; }
• По классу:.class { color: blue; }
• По идентификатору:{ color: green; }

Основные свойства CSS

• Цвет: color, background-color
• Шрифты:font-family, font-size, font-weight
• Рамки:padding, margin
• Цвет: border
• Выравнивание: text-align, vertical-align

Макеты

• Блочные элементы: <div>, <section>, <article>
• Гибкие макеты: display: flex;, display: grid;
• Позиционирование: position: relative;, position: absolute;, position: fixed;

ОСНОВЫ JS

JavaScript — это язык программирования, который используется для динамического взаимодействия с веб-страницей, создания анимаций, обработки событий и выполнения логики на клиентской стороне.

ВВЕДЕНИЕ В JS

JS добавляется в HTML с помощью тега <script>:
<script<console.log("Hello, world!"); </script>

ПЕРЕМЕННЫЕ

Объявление переменных:
let, const, var
Типы данных: строка ("text"), число
(42), булев тип (true, false),
массив ([1, 2, 3]),
объект ({name: "John", age: 30})

ОПЕРАЦИИ

Арифметические операторы:+, -, *, / ;
Сравнение: ==, !=, >, <, >=, <= ;
Логические операторы:
&& (и), || (или)

ЦИКЛЫ

for:
for (let i = 0; i < 5; i++) { console.log(i); }
while:
let i = 0; while (i < 5) { console.log(i); i++; }

УСЛОВНЫЕ КОНСТРУКЦИИ

if (age >= 18) { console.log("Вы совершеннолетний."); } else { console.log("Вы не совершеннолетний."); }

ФУНКЦИИ

function greet(name) { return "Привет, " + name; } console.log(greet("Мир"));

ОБРАБОТКА СОБЫТИЙ

Слушатель события: addEventListener document.getElementById("myButton").addEventListener("click", function() { alert("Кнопка нажата"); });

МАНИПУЛЯЦИИ С DOM (Object Model)

DOM — это интерфейс для взаимодействия с HTML-документом с помощью JavaScript.
document.getElementById("myDiv").innerHTML = "Новый текст";
